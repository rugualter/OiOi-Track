# OiOi-Track

Based on the original work of @FuzzyGrim (OiOi-Track) this is a completly new direction of the self hosted media tracker for movies, tv shows, anime, manga, video games, books, comics, and board games. Renamed OiOi-Track. This version hosts a lot of new functionalities, such as multilanguage support, TVDB source on medias, and much more, and more to come. This is still work in progress so expect a few bugs.
